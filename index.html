Sobre

Sou Jonatas, desde 2005 trabalho com desenvolvimento utilizando Java, PHP e entre outras tecnologias, frameworks e bibliotecas para desenvolvimento de sistemas para ambientes corporativos.

Atualmente, apaixonado pelo universo do Javascript - especificamente Node JS, React JS e React Native - venho criando microserviços e aplicações que consomem APIs.
